# SugarShield - Quick Start (5 Minutes)

## For Mac Users

### 1. Open Terminal
- Press `Cmd + Space`
- Type "Terminal"
- Press Enter

### 2. Navigate to the project
```bash
cd ~/Downloads/sugarshield
```

### 3. Install dependencies
```bash
npm install
```
This will take 2-3 minutes.

### 4. Start the app
```bash
npm run dev
```

### 5. Open in browser
Go to: **http://localhost:3000**

## You're done! 🎉

### Try these features:
- **Scan tab**: Use your camera to scan product barcodes
- **Upload tab**: Upload a photo of nutrition labels
- **Link tab**: Paste product URLs
- **/eval**: See the evaluation framework
- **/metrics**: View usage analytics

### Deploy to Vercel (Optional)
```bash
npm install -g vercel
vercel
```

See `SETUP_GUIDE.md` for detailed instructions.
